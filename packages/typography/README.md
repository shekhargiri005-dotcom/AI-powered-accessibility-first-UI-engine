# @ui/typography

Responsive text components: Heading, Text, and Caption.
Built with fluid scales and AI-optimized readability.

## Installation
```ts
import { Heading, Text, Caption } from '@ui/typography';
```

## Usage
```tsx
<Heading level={1}>Hello World</Heading>
<Text>This is optimized for reading.</Text>
<Caption>Footer note.</Caption>
```
