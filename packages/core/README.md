# @ui/core

Core UI primitives for the AI-powered UI engine. 
Features AI-adaptive behaviors, smooth morphing animations, and voice activation.

## Installation
```ts
import { Button, Card, Modal, Input } from '@ui/core';
```

## Usage
```tsx
<Button askAI onClick={() => alert('AI Triggered')}>Ask AI</Button>
<Input voiceActivation placeholder="Speak your command..." />
```
